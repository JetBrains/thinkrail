## Uppercase Labels

Standard uppercase label style for section headers, table headers, and UI labels.
Use the `.text-uppercase` utility class or the CSS custom properties:
- `--uppercase-size` (13px / `--font-body`)
- `--uppercase-weight` (500 / medium)
- `--uppercase-spacing` (0.5px)
