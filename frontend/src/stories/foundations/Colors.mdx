# Colors

Color tokens sourced from `src/styles/tokens.css` (accents, shared across themes)
and `src/styles/theme-*.css` / `themes.css` (semantic roles, redefined per theme).
Each swatch is a live `var(--token)` reference — change the **Theme** in the
toolbar and the semantic colors re-resolve.

## Accent palette

JetBrains brand accents from `tokens.css`. Identical in every theme.

<ColorPalette>
  {ACCENTS.map((c) => (
    <ColorItem key={c.token} title={c.token} subtitle={c.role} colors={{ [c.token]: `var(${c.token})` }} />
  ))}
</ColorPalette>

## Semantic colors

Surface, border and text roles from `theme-*.css` / `themes.css`.

<ColorPalette>
  {SEMANTIC.map((c) => (
    <ColorItem key={c.token} title={c.token} subtitle={c.role} colors={{ [c.token]: `var(${c.token})` }} />
  ))}
</ColorPalette>
