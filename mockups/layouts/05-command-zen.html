05 · Command Zen

Chrome-minimal, conversation-first: one reading column, everything else a keystroke away. Arc's hidden rails + a Linear-style ⌘K that teaches its own shortcuts.
  1. No dead bar. The 38px top row is gone — chrome collapses into a floating ⌘K capsule plus two corner buttons.
  2. Splits are commands. "Split chat right ⌘\" puts side-by-side chats one keystroke away — no chrome to hunt.
  3. Projects = a command, not a panel. "Switch workspace ⌘2" replaces the always-open left tree: zero idle width.
  4. Terminal unsqueezed. ⌘J opens a full-height edge overlay — the whole viewport when needed, a 3px strip when not.
  5. Files peek from the LEFT. Hover the edge (or ⌘B) and the tree slides in over the column, then gets out of the way.
  6. Tabs move by command too. "move pane ⌘⇧←→" in the palette footer — rearranging needs no drag targets.
NN/g caveat honored: opt-in expert mode; every hidden surface keeps a visible signifier (edge tabs + status-bar hints).
