01 · IDE Classic

The VS Code workbench shape — chats become first-class editor tabs inside splittable groups.
  1. Top bar works for a living. Brand, workspace switcher, ⌘K search/commands, model, connection and settings share one 34px strip — no dead row.
  2. Side-by-side is just a split. The center is editor groups; a chat sits next to a file (or another chat), each group with its own tab strip.
  3. Projects demoted to a switcher. The rarely-used 18%-wide panel becomes a top-bar pill (plus a rail icon) — switching costs a click, not width.
  4. Terminal gets the floor. A full-width bottom panel (~28%, maximize ⌃) with real tabs replaces the cramped right-column strip.
  5. Files sit left in a classic 240px Explorer with git-status colors, driven from the icon rail — where IDE users expect them.
  6. Tabs travel. Any tab drags between groups — shown mid-drag: a ghost “tokens.css” and a drop-to-split zone over the chat group.
