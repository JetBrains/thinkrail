08 · Dual Rail

The conservative evolution every major IDE converged on: files left behind an icon rail, the agent in a dedicated ~600px right dock, a full-width terminal below — and every dock relocatable.
  1. Top bar earns its 34px: workspace switcher and the ⌘K search/command pill live there — no more dead chrome.
  2. 2+ chats visible: the agent dock stacks its own chat tabs, and the ghost tab shows any chat promoted (⇱) into the editor area — side-by-side falls out of ordinary splits.
  3. Projects demoted: a rail icon + the workspace pill replace the always-open 18% panel for a rarely-used switch.
  4. Terminal breathes: a full-window-width bottom panel with one-click maximize (⌃).
  5. Files where eyes look: Explorer docks left, behind the Files rail icon — standard IDE muscle memory.
  6. Views are location-independent: every panel's ⋮ menu offers Move to Left / Bottom / Editor area / New window (the VS Code views model), so no dock position is ever forced.
Pattern: VS Code secondary sidebar · Zed agent dock
