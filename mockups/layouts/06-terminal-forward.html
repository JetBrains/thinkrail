The bottom panel graduates to a tmux-grade workspace: 48% tall, one-key splits, zoom, named layout presets, broadcast — and chats dock into it like any other tab.
  1. Top bar earns its 34px. Brand, workspace switcher, ⌘K palette, model + engine health — no dead chrome.
  2. Two chats at once — one up, one down. Chats are ordinary tabs: "Layout redesign" lives in the editor group while "Fix e2e flake" is docked in the terminal workspace.
  3. Projects leave the sidebar. The "thinkrail / main ▾" pill switches project + worktree from the top bar; no always-open panel.
  4. THE fix: terminal as workspace, not drawer. 48% of the column, ⌘D splits, ⇶ broadcast, ▤ layouts (tall · fat · grid), ⇱ move to editor area, ⤢ maximize (⌘⇧J) — the tmux/kitty checklist.
  5. Explorer on the left, fixed 220px, where IDE muscle memory expects files.
  6. Every tab is portable. Chat, file and terminal tabs drag between both strips — the ghost + "Drop to dock chat here" show a chat landing beside a terminal pane.
