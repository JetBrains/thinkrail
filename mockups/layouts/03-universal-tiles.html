03 · Universal Tiles

Every tab — file, chat, terminal — lives in one dockable tile system. Drag a tab anywhere: center merges, edges split. Tiles float, pop out to windows, and arrangements save as presets.
  1. Working top bar. ⌘K search/commands + a Layout preset pill ("Deep Work") that saves and restores serialized arrangements.
  2. Many chats at once. Two chats run side by side in independent tile groups — drag any grid into existence; a terminal tab co-lives with a chat.
  3. Projects shrunk to a pill. Workspace switching moves to the top bar; no always-open 18% column.
  4. Terminal unsqueezed. Terminals dock wide at the bottom, float as mini windows, pop out (⧉) to a browser window, or maximize (⤢).
  5. Files where you want them. The files pane is just a tile — this user docked it left. Any pane docks anywhere.
  6. Tabs are free. Mid-drag, the 5-zone drop overlay appears on the target pane: center = merge into the group, edges = split ("Split down" shown hot).
Pattern: Dockview · JupyterLab · VS Code editor groups
