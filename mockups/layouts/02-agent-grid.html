02 · Agent Grid

Chat-first mission control: a preset grid of live agents, status-first tiles — and "needs your input" is the loudest thing on screen.
  1. Top bar earns its row. Preset layouts (⌘1–4), ⌘K and New chat replace the dead space; maximize-one (⏎) beats free-form docking.
  2. Side-by-side chats ARE the layout. 1 / 1×2 / 2×2 / 1+2 presets tile the whole center — no single-tab bottleneck.
  3. Projects shrink to a pill. The rarely-touched tree becomes a dropdown; its ~18% column goes back to chats.
  4. Terminals get the full width. Terminals · Files · Changes dock in a ⌘J drawer — full-width and maximizable when opened.
  5. Changes peek into each chat. Per-tile diff chips cover the common glance; the full tree waits in the drawer.
  6. Tiles rearrange like tabs. Drag a tile header across the grid gutter — drop to swap panes.
