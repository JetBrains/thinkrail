rookery · architecture

Mobile-first React Native client with offline-first SQLite, syncing to a small FastAPI/Postgres backend.

Stack

  • Client: React Native (Expo), TypeScript.
  • On-device storage: SQLite (expo-sqlite).
  • Backend: FastAPI on Python 3.11, async.
  • Server storage: Postgres + Alembic migrations.
  • Media: CDN-fronted blob store; thumbnails on-device.

Modules + added

  • capture — one-tap log, photo, location, species guess
  • sync — offline queue, conflict resolution, backend sync
  • lifelist — chronological log, map view, count, filters
